qoifde

Currently complete

Encoder and decoder that can perform basic operations on QOIF images. Conversion to and from raw RGB/RGBA is possible. Was pleasently simple to implement. I have no connection to the format creators. if you are interested in learning more about the format see their website.

dank_json

Not yet Complete

Yet another library for working with JSON in C... what makes this one special? The programmer is not required to call malloc and free on the nodes they create because all allocation is done behind the scenes. The lifetimes of nodes can be terminated on a per-node, per-file, or whole world basis. Accessing subnodes of the root is done by supplying execl and execv style parameters. Try it out.
